# langchain_practice
This is a practice Course for LangChain

# Module Highlights
1. LLMs

Examples showing integration with OpenAI, HuggingFace, etc.

2. ChatModels

Chat-oriented LangChain prompts & conversations.

3. EmbeddingModels

Embedding generation and practical use cases.

4. LangChain Prompts

Template prompt engineering & chaining.

5. Structured Output

Output structs and formatting using modern LangChain.

6. Output Parsers

Parsable outputs, including Pydantic, JSON, and custom parsers.

7. Chains

Demonstrations of sequential and parallel chains.

8. Document Loaders

Loading documents from local files and web sources.

9. Text Splitters

Text splitting strategies, chunking and semantic splitters.
